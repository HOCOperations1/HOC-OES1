🤖 How Auto-Detection Works
Every 30 seconds, this dashboard scans the Production Schedule + Production Supervisor floor updates and infers changeovers automatically:
  • Same formula + same size → label/lot change, skipped (not a real changeover)
  • Same formula, different size/pack → CO[60m] medium changeover
  • Different formula entirely → CO[90m] full changeover
Actual duration is inferred when PS marks the previous batch COMPLETED and the next batch RUNNING — the time gap = actual changeover time. Overruns >125% fire critical events automatically.
Manual SMED tracking is kept for kaizen workshops where teams want to track individual SMED tasks step-by-step.
Note: Auto-detect reflects the current uploaded schedule. For historical records and YTD analysis, see the Patch OEE upload (Line Scorecard tab).
