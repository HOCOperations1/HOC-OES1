How to use HOC-OES correctly
Click any tool above to open it directly. Each tool has a ← HOC-OES button to return here.
All tools share the sync bus automatically — updates from Pre-Weigh appear in Compound Scheduler, Leadership Tower, and everywhere else.

For full sync across multiple devices: host this folder on SharePoint or run python -m http.server 8000 in the folder, then open http://localhost:8000/index.html.
