How 2-Way Sync Works
🟢 Production Supervisor
Logs: batch status, actual cases, downtime. Updates visible instantly in Compound Scheduler and Quality Lab when open in the same browser.
🔵 Compound Scheduler
Logs: compound status, tank, lbs. Sees production status from Production Supervisor. Updates visible in Quality Lab.
🩵 Quality Lab
Logs: QC status, pH, viscosity, release date. Sees both production and compound status. QC release triggers visible in all tools.
All three tools share one sync key in the browser. Open any two in separate tabs — updates from one appear in the other on next page load or refresh.
