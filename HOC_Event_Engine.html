Setup Steps
  1. Open Power Automate (flow.microsoft.com) and sign in with your House of Cheatham account.
  2. Click + Create → Scheduled cloud flow.
  3. Name: HOC-OES Event Engine. Run every 30 minutes (or every hour — frequent enough to catch the 11 AM and 4 PM windows).
  4. Add an HTTP action: GET https://yemtpvrumqvbrzrpwnyy.supabase.co/rest/v1/hoc_events?sent_at=is.null with headers apikey and Authorization: Bearer ... (use the anon key from the README).
  5. Loop over the returned JSON array. For each event:
    • Look up the matching config row from hoc_event_config.
    • Check the schedule field — if 5pm+12am, only send when current time is 17:00–17:30 or 00:00–00:30. If 11am+4pm, only send when current time is 11:00–11:30 or 16:00–16:30.
    • If immediate, send right away.
    • Use the Outlook 365 → Send an email (V2) action with the config's recipients, subject template, and event payload.
    • If pdf_url is present, fetch the PDF and attach it.
    • POST back to Supabase to set sent_at = utcNow() and increment send_attempts.
  6. For QC Releases on 5pm+12am schedule, group all pending QC releases into ONE summary email with all PDFs attached.
Flow Template Download
A pre-built Power Automate flow template ships alongside this dashboard as HOC_OES_EventEngine_Flow.json. Import it via Power Automate → My flows → Import → Import Package.
After import, you'll be prompted to:
  1. Connect your Outlook 365 account (for sending email)
  2. Confirm the Supabase URL + anon key are set as Secure Inputs in the flow's HTTP actions
  3. Turn the flow on
Manual fallback (works without Power Automate)
Until the Power Automate flow is live, click "📧 Send All Pending Now" on the Event Feed tab to compose a single email containing every pending event. This opens your default email client with the body pre-filled and recipient list applied. You then review and click send.
